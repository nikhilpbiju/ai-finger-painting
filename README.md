# ai-finger-painting
FingerDraw is a real-time computer vision application that allows users to draw on a digital canvas using hand gestures captured through a webcam. The system tracks the position of the index fingertip and converts its movement into drawing strokes, effectively simulating a virtual pen. The canvas is overlaid on the live video feed, creating an intuitive and interactive experience without the need for physical input devices.

The application supports multiple drawing colors, an eraser mode, canvas clearing, and saving drawings as image files. It demonstrates the practical use of hand tracking and real-time image processing for building touchless human-computer interaction systems.

Tech Stack

  ->Python
  
  ->OpenCV  
  
  ->MediaPipe
  
  ->NumPy

Controls

1 → Red

2 → Green

3 → Blue

E → Eraser

C → Clear canvas

S → Save drawing

ESC → Exit
